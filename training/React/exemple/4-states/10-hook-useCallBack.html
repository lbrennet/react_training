Hooks

Les Hooks sont une nouveauté de React 16.8.
lorsque React met à jour les composants, une nouvelle fonction peut déclencher
inutilement un rendu.
Pour cela React a prévu un HOOK pour les fonctions callback
useCallback renverra une version mémoïsée de la fonction de rappel
qui changera uniquement si une des entrées a changé.
